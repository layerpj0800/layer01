스켈레톤 및 로직


/frontend (Next.js + React Query + TailwindCSS + shadcn/ui)
  ├─ app/                   # App Router 기반 페이지
  │  ├─ (public)/           # 공개 페이지(홈, 채널 리스트, 채널 상세)
  │  ├─ (auth)/             # 회원가입/로그인
  │  └─ (dashboard)/        # 구독자·호스트 대시보드
  ├─ components/            # UI 컴포넌트
  ├─ hooks/                 # React Query, 유틸 훅
  ├─ lib/                   # API 클라이언트, 인증/쿠키 유틸
  ├─ styles/                # 글로벌 스타일·Tailwind 설정
  └─ public/                # 정적 리소스

/backend (FastAPI + SQLAlchemy + Alembic)
  ├─ app/
  │  ├─ api/                # /api/v1 엔드포인트
  │  ├─ models/             # SQLAlchemy 모델
  │  ├─ schemas/            # Pydantic 스키마
  │  ├─ services/           # 비즈니스 로직
  │  ├─ auth/               # JWT, 인증/권한
  │  ├─ workers/            # Celery/RQ 작업자
  │  └─ core/               # 설정, DB 연결, 미들웨어
  ├─ migrations/            # Alembic 마이그레이션
  └─ tests/                 # Pytest 기반 테스트

/infrastructure
  ├─ docker/                # Dockerfile, docker-compose
  ├─ scripts/               # 배포/CI 스크립트
  └─ terraform/             # 클라우드 인프라 코드(선택)
